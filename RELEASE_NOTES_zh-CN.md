# 仓库更新说明

## 本次同步内容

### 1. 心情评估改为角色自我视角

- 初始心情评估不再以旁观者或裁判视角分析日志。
- 实时对话情感评估会读取当前角色的基础 system prompt，并要求模型站在“肥牛本人”的角度判断心情增减。
- 这样可以更准确地区分“嘴硬式打闹”和“真正被冷落/被敷衍”的差别，避免无参考意义的第三方打分。

### 2. 新增游戏场景窗口核对

- 当视觉模型把当前画面识别为游戏时，可额外结合前台窗口标题与进程进行二次核对。
- 这能降低 IDE、开发工具、某些深色界面被误判成游戏的概率。

### 3. 配置示例同步更新

- 公开仓库保留了可参考的心情评估提示词示例。
- 公开仓库已移除 API Key、本地日志路径、私有主动搭话长提示词等敏感信息。
- `plugin_config.json` 中新增了 `scene_window_cross_check_enabled` 配置项，用于控制窗口核对开关。
